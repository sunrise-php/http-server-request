## Installation

```
composer require sunrise/http-server-request
```

## How to use?

```php
use Sunrise\Http\ServerRequest\ServerRequestFactory;

$request = ServerRequestFactory::fromGlobals();

// just use PSR-7 methods...
```

## Test run

```bash
php vendor/bin/phpunit
```
